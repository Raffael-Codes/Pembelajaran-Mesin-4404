# 📘 Python Basic Data Structures

## 📌 Deskripsi

Project ini berisi contoh penggunaan struktur data dasar di Python, seperti:

* Tuple
* List
* Dictionary

Selain itu, project ini juga menampilkan operasi dasar seperti:

* Menampilkan data
* Mengubah nilai (update)
* Iterasi (looping)
* Slicing (reverse)

---

## 📂 Isi Project

Beberapa implementasi yang terdapat dalam project ini:

* Pembuatan dan penggunaan **tuple**
* Pembuatan dan penggunaan **list**
* Pembuatan dan penggunaan **dictionary (kamus)**
* Update nilai pada dictionary
* Looping menggunakan `for`
* Slicing untuk membalik urutan data
